claws-mail-indicator by Marek Habersack

Claws Mail plugin to show new mail notifications using the Ubuntu indicators and Unity

claws-mail-indicator is a small plug-in for the Claws Mail MUA which implements Ubuntu-style notifications. The plug-in integrates with the Ubuntu indicators making the notifications blend in nicely with the Ubuntu desktop. The Messaging indicator entry for Claws Mail includes the same menu options as those implemented by the TrayIcon plug-in. The plug-in was developed and tested on Ubuntu 11.04 (Natty).

Dependencies

  • libindicate
  • glib2
  • claws-mail
  • libnotify [optional]
  • libunity [optional]

Install

For now you need to install all the dependencies and compile the plug-in from source. After that you can load it in Claws Mail directly from the src/.libs directory or, alternatively, put it in the /usr/lib/claws-mail directory.
